Editing & Compositing

Edited in DaVinci Resolve. For key moments in the script I generated background visuals using Google Flow to make the story more visually engaging — floating numbers, baby items drifting through space, wifi waves. Prompts were written with the help of Claude. All elements were keyed and composited directly in Resolve's Fusion page.

DaVinci Resolve Fusion Google Flow Claude

Sound

Background music pulled from my own personal library — faster to grab something I already have access to than searching a stock platform, though I use Artlist regularly when the project calls for it. SFX are a mix of my own collection and freesound.org.

Personal library freesound.org

Captions

Generated using DaVinci Resolve's built-in AI transcription. It's the fastest caption workflow I've found that stays entirely inside one program — no quality loss, no round-tripping through CapCut or Premiere. The style here is the default; caption design in Resolve is fully customizable, and I'd build a proper branded preset on day one if we work together. Set it up once, have it forever.

DaVinci Resolve AI

Color

Light correction pass — boosted saturation slightly, balanced the clips against each other, kept the organic handheld feel intact.

DaVinci Resolve Color

What I'd do differently

Custom caption styling would be the first thing. A couple more VFX moments if I had more tokens.

As a cinematographer, I'd also approach the shoot itself differently. If given the chance to brief the camera operator beforehand, I'd set up the shots with post in mind — framing, lighting, and movement choices that make compositing and color work significantly cleaner and faster. That conversation at the start saves a lot of time at the end.

Otherwise I'm happy with where it landed — curious to hear what the team thinks.
